Основи CSS

У цьому єпізоді молоді падавани почнуть знайомство з каскадними таблицями стилю іменованими CSS. На них чекає захоплююча подорож у мові, яка відповідає за опис зовнішнього вигляду HTML-документа. Переважна більшість сучасних веб-сайтів працюють на основі зв'язку HTML+CSS.

CSS використовується для визначення стилів (правил) оформлення документів - включаючи дизайн, верстку та варіації макету для різних пристроїв та розмірів екрану. У мови CSS відносно простий синтаксис. Спочатку прописується селектор він вибирає конкретний елемент на сторінці. Потім після фігурних дужок вказуються властивості зі значеннями - між ними ставиться двокрапка. Самі властивості відокремлюються один від одного крапкою з комою.

Варіанти підключення CSS:

  • Глобальнний CSS
  • Зовнішній CSS
  • Внутрішній CSS

Глобальнний CSS

Глобальний CSS розміщується в контейнері "head" конкретної сторінки. При такому варіанті підключення класи та ідентифікатори (ID) можуть бути використані для звернення до CSS коду однак, вони будуть активні тільки на цій конкретній сторінці.

Зовнішній CSS

Можливо найзручніший варіант для підключення CSS до вашого сайту, це його прив'язка до зовнішнього .css файлу. У цьому випадку всі зміни, зроблені в зовнішньомуCSS файлі, будуть в цілому відображатися на вашому сайті.

Внутрішній CSS

Внутрішній CSS використовується для конкретного HTML тега. Атрибут "style" використовується для налаштування цього тега. Цей варіант підключення CSS не є рекомендованим, тому що в цьому випадку потрібно налаштовувати кожен тег HTML окремо.
